CRITICAL WARNING: HOW TO MARK BOUNDARIES

It is crucial to mark the points in the correct order to get an accurate area.

DO THIS (Correct Way):
  • Mark the points in a SEQUENCE (Circle-wise).
  • Start at one corner, then go to the immediate next corner along the boundary line (A ‚Üí B ‚Üí C ‚Üí D).
  • Result: This forms a clean, closed shape (Polygon).
DO NOT DO THIS (Wrong Way):
  • Do NOT mark diagonally (Criss-Cross).
  • Example: Do not mark Top-Left corner, then jump to Bottom-Right corner.
  • Why? This causes the boundary lines to INTERSECT (cut each other), creating a "bow-tie" or "hourglass" shape. The app cannot calculate the area of self-intersecting shapes correctly.

Once you have formed a proper closed shape, click the Calculate button.

CUSTOM UNIT CALCULATION & BASE UNIT

At the bottom of the result screen, you will see the "Custom Unit" section. This is for converting the area into local units (like Bigha, Guntha, Kanal).

How to use it correctly:

  1. Look at the Total Area displayed above.
  2. Identify the Base Unit. The values shown in the specific cards (e.g., the value below "Area Calculated Successfully" in Square Meter or other unit) represent your Base Unit.
  3. In the Custom Unit section, ensure the Dropdown Menu is set to the same unit you are looking at.
    • Example: If you are converting the 'Square Meter' value, set the Dropdown to 'Square Meter'.
    • Example: If you are converting the 'Square Feet' value, set the Dropdown to 'Square Feet'.
  4. Enter your local unit value (e.g., 1 Bigha = X Sq Meter).
  5. Press Calculate to get the result in your custom unit.

‚ö†Ô∏è Mistake to Avoid: Do not leave the base unit and custom unit dropdown as 'Square Feet' if you are entering a 'Square Meter' value. The units must match!
